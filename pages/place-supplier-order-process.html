Place Supplier Order process

 

The purpose: To add a new Supplier Order to the BagIT database.

 

How to: The User clicks on the Supplier tab on the navigation bar and clicks the Supplier Orders link. The User will click on the Add button on the top right of the Supplier Order home page. The Add Supplier form will be displayed, and the User will fill in the details of a Supplier.
