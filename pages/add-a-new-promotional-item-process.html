Add a new Promotional item process

 

The purpose: To add a new Promotional item to the BagIT database.

 

How to: The User clicks on the Categories tab on the navigation bar and clicks the Promotional items tab. The User will click on the Add button on the top right of the Promotional items home page. The Add Promotional items form will be displayed, and the User will fill in the details of a Promotional items.
