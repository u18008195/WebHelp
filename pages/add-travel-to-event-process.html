Add Travel to Event process

 

The purpose: To add a Travel to the Event.

 

How to: In the Event Home page, the User clicks on the Travels dropdown on the left side. The User will click on the plus button to add the Travel item to the event Alternatively search for a Travel item in the Search textbox in the dropdown tab or the user alternatively click the “Add Travels” button which will redirect to the Travels home page. Refer to the Add Travel item process. A screen will display called “Select Item Day”, the user selects the day for the Travel item. The user can view the Travel Item in the “Event details” dropdown, on the right side of the page.
