Update a Travel item process

 

The purpose: To update a Travel item to the BagIT database. This will allow the User to have access to the system and system functionalities allocated to users.

 

How to: The User clicks on the Categories tab on the navigation bar and clicks the Travel tab. The User will click on the Update button in the table row of the Travel item. The Update Travel form will be displayed, and the User will fill in the details of a Travel.
