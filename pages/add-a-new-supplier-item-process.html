Add a new Supplier item process

 

The purpose: To add a new Supplier to the BagIT database.

 

How to: The User clicks on the Supplier tab on the navigation bar and clicks the View Supplier link. The User will click on the Add button on the top right of the Supplier home page. The Add Supplier form will be displayed, and the User will fill in the details of a Supplier.
