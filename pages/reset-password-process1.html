Reset Password process

 

The purpose: To Login to the BagIT system.

 

How to: The User opens the BagIT system. The User will then click the forgot password hyperlink, which will prompt the user to the Account Recovery page. The User will then enter their relevant details into the appropriate textboxes, which will allow them to reset their password.
