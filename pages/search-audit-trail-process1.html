Search Audit Trail process

 

The purpose: To search for activity in the Audit Trail on the BagIT system.

 

How to: The Administrator clicks on the Users tab on the navigation bar. The User will click on the View Audit Trail button on the top right of the User home page. The Administrator will click on the search textbox above the table on the Audit Trail home page. The administrator must type a correct attribute from the Audit Trail and the table will filter and display the item being searched.
