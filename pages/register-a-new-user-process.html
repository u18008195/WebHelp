Register a new user process

 

The purpose: To register a new User to the BagIT database.

 

How to: The Administrator clicks on the Users tab on the navigation bar. The User will click on the Add button on the top right of the User home page. The Register new User form will be displayed, and the Administrator will fill in the details of the User.
